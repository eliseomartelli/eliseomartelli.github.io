Nebula is a peer-to-peer software that allows machines that are located on different networks to talk to each other.  
The communication between hosts is encrypted uusing the [Noise Protocol Framework](https://noiseprotocol.org/) (the same used by Signal).

# Nebula Vocabolary

lighthouse
node

# Configuring Nebula

spin up a lighthouse node. A little 5$ Droplet on [DigitalOcean](https://m.do.co/c/33e2f0a1e231) will do.
generate ca
generate key for node(s)
copy ca to our lighthouse
copy keys to other nodes
config.yml
